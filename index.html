Help Section

Welcome to Maniac OS. This is an amazing Operating System which is capable of handling more than one scheduling algorithms. Moreover, you can even change the size of lists, probability to create a new process, and more! In order for you to understand it a little bit more, you have this helpful section.

Parameters values

  • Probability: [0, 100]
  • Quantum: [2, 100]
  • Average CPU: [1, 100]
  • Average IO: [1,CPU)
  • Hold, ready, waiting: [1, 50]

Rules

  • First of all you need to select an speed and algorithm from the parameters list.
  • To make a succesful change of any parameter, first you need to stop or pause the simulation. Then click Play again.
  • Almost any parameter can be modified, except one: the probability of a process to require IO or not. It is all the time 50/50.
